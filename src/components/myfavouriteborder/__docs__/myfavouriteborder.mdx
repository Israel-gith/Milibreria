# MyFavouriteBorder
Componente que personaliza el IconButton y FavoriteBorderIcon de MUI.

#### Example - Default
<Canvas of={MyFavouriteBorder.Default} />

## Usage
```typescript
import { MyFavouriteBorder } from 'milibreria';

function Example() {
  return (
    <MyFavouriteBorder
      color="blue"
      disabled={false}
      onClick={() => alert("Favourite Border Clicked")}
    />
  );
}

export default Example;
