# MyButton
Este es el componente de botón personalizado, con opciones de personalización de color, tamaño, y comportamiento en hover.

#### Example - Default
<Canvas of={MyButton.Default} />

## Usage
```ts

import { MyButton } from 'milibreria';

function Example() {
  return (
    <MyButton
      text="Default Button"
      textColor="white"
      bgColor="blue"
      borderColor="black"
      borderWidth={1}
      hoverTextColor="yellow"
      hoverBgColor="darkblue"
      size="medium"
      disabled={false}
      onClick={() => alert("Default Button Clicked")}
    />
  );
}

export default Example;
