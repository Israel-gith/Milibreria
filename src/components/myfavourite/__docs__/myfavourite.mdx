# MyFavourite
Componente que personaliza el IconButton y FavoriteIcon de MUI.

#### Example - Default
<Canvas of={MyFavourite.Default} />

## Usage
```ts
import { MyFavourite } from 'milibreria';

function Example() {
  return (
    <MyFavourite
      color="red"
      disabled={false}
      onClick={() => alert("Favorite Clicked")}
    />
  );
}

export default Example;
