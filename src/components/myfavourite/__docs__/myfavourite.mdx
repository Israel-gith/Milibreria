# MyFavourite

Componente de icono con diferentes especializaciones.

#### Example

<Canvas of={MyFavourite.Primary} />

## Usage

```ts
import {MyFavourite} from 'milibreria';

function Example(){
    return (
        <MyFavourite
         iconcolor: 'red',
         disabled: false,
         onClick: () => alert("MyFavourite primary"),
        />
    );
};

export default Example;
```
