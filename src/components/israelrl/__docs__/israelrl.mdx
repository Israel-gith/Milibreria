# IsraelRL

Componente AppBar con cambio de texto y colores.

#### Example

<Canvas of={IsraelRL.Primary} />

## Usage

```ts
import {IsraelRL} from 'milibreria';

function Example(){
    return (
        <IsraelRL
         text: "Barra Primary" ,
         textcolor: "black",
         barcolor: "blue"  , 
         onClick: () => alert("IsraelRL Primary"),
        />
    );
};

export default Example;
```
